- Open [discord devloper](https://discord.com/developers/applications)

- ![GOTO BOT](https://user-images.githubusercontent.com/40953479/105614356-1a8a1a00-5dc9-11eb-8c9b-b45e2c418afe.png)
- ![ADD A NEW BOT](https://user-images.githubusercontent.com/40953479/105614366-3f7e8d00-5dc9-11eb-8277-5711bafb8ee4.png)
- ![JUST DO IT!](https://user-images.githubusercontent.com/40953479/105614387-63da6980-5dc9-11eb-815b-22c2418c015a.png)
- ![GOTO OAUTH](https://user-images.githubusercontent.com/40953479/105614417-ac922280-5dc9-11eb-9f2e-3809abe80fae.png)

- Open a Command Prompt
- (OPTIONAL) Type `npm i sharkdiscordbot -g` to install it, it will store the entered token so that you doesn't need to enter the token again
- Type `npx sharkdiscordbot -t "your token"` (INSTEAD OF `your token` you paste the token)
- It boot up and connect with your discord channel, if you have installed it (THE OPTIONAL THING) you can now start it after closing with `npx sharkdiscordbot`
