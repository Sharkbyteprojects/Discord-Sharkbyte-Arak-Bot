# youtubeMetaData

Promise based

- Installation: `npm i youtube-meta-data --save`
- Usage: `require("youtube-meta-data")("https://www.youtube.com/watch?v=WhWc3b3KhnY").then((info)=>{console.log(info);}, (err)=>{/*ERRORHANDLING*/})`
